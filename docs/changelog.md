# 更新日志

## v0.2.3

- 数据库连接能力继续增强：MySQL 连接新增 `charset` 与 `collation` 参数，PostgreSQL 连接可靠性与延迟表现进一步优化。
- 交互与文案修复：修正标题栏交互支持逻辑，并统一 `ConnectionType` 标签展示。
- 工程与资源更新：优化 macOS 图标生成流程，并同步更新 README 与界面截图资源。

## v0.1.9

- 数据库表设计器和 SQL 预览相关体验继续增强。
- 数据库树与数据操作流补充了更多高频交互细节。

## v0.1.8

- 围绕数据库和连接相关能力持续修复与打磨。
